Grab MinGW-w64 with GCC 12 or later and CMake 3.20. Ninja or Make both work as the backend. From the repo root do cmake -B Build -G "MinGW Makefiles" -DCMAKE_BUILD_TYPE=Release then cmake --build Build. One thing that trips people up is linker errors about WinMain. That usually means you left out the Release build type flag, because Debug mode pulls in extra console dependencies on MinGW and the entry point gets confused.
