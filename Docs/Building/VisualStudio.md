You need Visual Studio 2019 or 2022 with the Desktop development with C++ workload installed. CMake 3.20 is also required. From the repo root run cmake -B Build -G "Visual Studio 17 2022" -A x64 then cmake --build Build --config Release. For 32-bit builds swap the architecture flag to Win32 and use a different output directory like Build32 so the files do not collide. The built DLLs end up under Build/Bin in subfolders for each emulator target. Visual Studio puts Debug and Release builds in separate directories under that path, so if you built Debug look under the Debug subfolder, not Release.
