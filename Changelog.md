Version 1.0.0 released June 22, 2026. This is the first public release of the Unified Ubisoft Emulator Suite. It ships four emulators: EmuR1 covers Uplay R1 with all exports, EmuR2 covers Uplay R2 the same way, EmuUPC handles Ubisoft Connect and its Demux protocol, and EmuOrbit targets the legacy Orbit DRM and was tested against Driver San Francisco. The shared LibUbi core library bundles INI parsing, crypto, networking, protocol buffers, API hooking, and logging into a single dependency. A unified Ubi.ini drives configuration for all four emulators. The build system uses CMake and works with MSVC, MinGW, and Clang. There is a small tools suite with a config converter, an export validator, and a game ID resolver. CI covers all three compilers.

A few things are not perfect yet. EmuOrbit LAN multiplayer has not been tested and might have packet ordering problems. The Docker setup only produces 64-bit DLLs because x86 cross-compilation requires a separate toolchain. Debug mode logs heavily and older CPUs can stutter in game as a result. The UPC overlay is a stub that passes calls through without drawing anything.
